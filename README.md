# C# Tetris Oyunu

## Kontroller:  
R -> Yeniden Başlat  
Up -> Döndür  
Down -> Hızlandır  
Right -> Sağa Git  
Left -> Sola Git  
Space -> Sert Bırak  
  
Blok Yerleştirme : 10 Puan  
Satır Tamamlama : 100 Puan
